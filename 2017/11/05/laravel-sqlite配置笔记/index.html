开启服务器

下面配置sqlite数据库

修改config文件夹下的database.php文件，去掉env(‘DB_DATABASE’)语句

'sqlite' => [
          'driver' => 'sqlite',
          'database' => database_path('database.sqlite'),
          'prefix' => '',
      ],

修改.env文件，把DB_CONNECTION改为sqlite，DB_DATABASE改为本地数据库的路径
